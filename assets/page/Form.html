proceso de trabajo

Primero se realiza el molde con las medidas requeridas previamente, se presenta el molde encima de la tela para realizar el corte utilizando alfileres para sostener ambas piesas. Luego unis las piezas recortadas en la remalladora uniendo los extremos correspondientes y verificando que la tela no se haya dañado durante el corte, una vez la prenda esta cosida con la remalladora se le realizan terminaciones con la collareta dandole bordes esteticos.
